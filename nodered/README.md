# Node-RED

Denne mappen skal inneholde eksporterte flows og forklaringer for Node-RED-delen av systemet.

Planlagt innhold:
- eksport av gjeldende alarmflow
- beskrivelse av MQTT in -> filtrering -> rate limiting -> ntfy send
- notat om bruk av container-navn internt i Docker
- dokumentasjon av hvilke noder som er testnoder og hvilke som er produksjonsnære

Viktig:
Ikke legg inn klartekst-passord, tokens eller andre hemmeligheter i eksporterte flows.
