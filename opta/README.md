# Arduino Opta

Denne mappen skal inneholde kode og dokumentasjon for Opta-delen av systemet.

Planlagt innhold:
- sketch eller PLC-relatert kode som faktisk er i bruk
- beskrivelse av hvilke innganger som overvåkes
- dokumentasjon av hvilke MQTT-topics som publiseres
- notat om payload-type, spesielt at DI1 er verifisert som boolean i Node-RED-løpet

Viktig:
Ikke commit WiFi-passord, MQTT-passord, API-nøkler eller annen sensitiv konfig direkte i kildekoden.
